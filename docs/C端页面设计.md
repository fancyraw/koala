## C端页面说明
#### 首页
```
┌──────────────────────────────────────┐
│ 顶部导航：App图标 + 品牌名 + 购物车角标   │
├──────────────────────────────────────┤
│ Banner 轮播区（限时特惠标签）            │
├──────────────────────────────────────┤
│ 领券横条：满100减10 | 新人立减5元  →     │
├──────────────────────────────────────┤
│ 品类专区（图标网格 + 查看全部）           │
├──────────────────────────────────────┤
│ 热销爆款（横向滑动卡片列表 + 查看全部）   │
├──────────────────────────────────────┤
│ 精选推荐（竖向左图右文列表）              │
└──────────────────────────────────────┘
```

| 页面 / 模块 | 入口 / 触发                       | 内容结构                                                     | 用户操作与系统反馈                                        | 关键规则                                         |
| ----------- | --------------------------------- | ------------------------------------------------------------ | --------------------------------------------------------- | ------------------------------------------------ |
| 首页        | App 启动默认页 / 底部 Tab「首页」 | 顶部导航、Banner、领券横条、品类专区、热销爆款、精选推荐     | 点击 Banner/品类/商品跳转对应页；快速加购角标+1           | 品牌名和App图标从后台「内容管理」读取；购物车角标数量加购后 +1 |
| Banner 区   | 首页默认展示                      | 大图 + 限时特惠标签                                          | 点击跳转商品列表页                                        | 数据来源：后台「内容管理-Banner」，图片/跳转链接/标签均为后台配置；图片异步加载，加载前展示占位 |
| 领券横条    | Banner 下方，首页默认展示          | 横向展示当前可领券（最多露出2-3张），右侧箭头                | 点击券直接领取；点击箭头进入领券中心页（全部可领券列表）   | 数据来源：后台「营销管理-优惠券」中状态为进行中的券；无可领券时隐藏该区域 |
| 品类专区    | 首页品类区                        | 品类图标+名称网格（品类数量可配置） | 点击任意品类跳转商品列表页并传递品类ID，按品类过滤；点击「查看全部」跳转列表页展示全部 | 数据来源：后台「商品管理-分类管理」 |
| 热销爆款    | 首页热销区                        | 横向可滑动商品卡片（商品图/标签/名称/已售数量/价格/加购按钮） | 点击卡片跳转详情；点击「+」加购按钮，购物车角标+1 | 数据来源：按销量排序取 Top N；商品标签从后台「商品管理」标签字段读取 |
| 精选推荐    | 首页底部                          | 竖向左图右文布局商品列表                                     | 点击卡片跳转商品详情页                                    | 数据来源：后台「商品管理」中标记为「推荐」的商品，无推荐商品时展示全部商品按上架时间倒序 |

#### 商品列表页

```
┌──────────────────────────────────────┐
│ 顶部导航：返回 + xx商城               │
├──────────────────────────────────────┤
│ 品类筛选栏（横向可滑动，选中高亮）        │
├──────────────────────────────────────┤
│ 排序栏（5种排序）+ 筛选按钮             │
├──────────────────────────────────────┤
│ 商品网格（双列瀑布流）                   │
│ [卡片: 主图 | 标签 | 名称 | 已售数量]   │
│ [现价 | 划线原价 | 加购按钮]            │
└──────────────────────────────────────┘
```

| 页面 / 模块 | 入口 / 触发                                      | 内容结构                               | 用户操作与系统反馈                                           | 关键规则                                         |
| ----------- | ------------------------------------------------ | -------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------ |
| 商品列表页  | 首页点击 Banner/品类/热销「查看全部」；Tab「发现」 | 品类筛选栏、排序栏、双列瀑布流商品卡片 | 点击筛选项：高亮并刷新列表；点击排序：列表重新排序；点击卡片：跳转详情；点击加购：购物车角标+1 | 从首页品类进入时默认选中对应品类；品类筛选与排序可组合使用；无结果时进入空状态 |
| 空状态      | 筛选结果为空                                      | 空状态图标 + 「暂无匹配商品」文字      | 无操作（其余区域保持）                                       | 触发条件：当前品类无匹配商品                      |

| 页面 / 模块              | 入口 / 触发               | 内容结构                                                     | 用户操作与系统反馈                                           | 关键规则                            |
| ------------------------ | ------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ----------------------------------- |
| 领券入口                 | 商品详情页价格区下方      | 展示当前商品可用的优惠券，如「满100减10」                      | 点击直接领取，已领取则显示「已领取」                          | 划线价为商品原价/市场参考价，现价为商品售价，券为额外优惠，三者独立 |
| 规格选择区               | 商品详情页默认展示        | 规格卡片列表：250g尝鲜装（¥49）/ 500g标准装（¥88），后台可配置，最多4个 | 点击规格卡片：选中态边框高亮，价格区实时切换为对应规格价格   | 默认选中「500g标准装」              |
| 数量步进区               | 商品详情页默认展示        | 减量按钮（-）+ 数量显示 + 加量按钮（+）                      | 点击加/减按钮：数量变化，价格不联动（按单价×数量在结算时计算） | 最小值为 1，数量为 1 时减量操作无效 |
| 底部操作栏               | 固定浮层，始终可见        | 购物车图标 + 加入购物车按钮 + 立即购买按钮                    | 购物车图标：跳转购物车页；加入购物车：留在当前页，购物车角标+1；立即购买：进入确认订单页 | 详情页不显示底部 Tab Bar            |
| 商品详情页不显示 Tab Bar | 路由配置 showTabBar=false | —                                                            | —                                                            | 仅商品详情页无 Tab Bar              |

#### 购物车页

```
┌──────────────────────────────────────┐
│ 顶部导航：返回 + 购物车（N件）           │
├──────────────────────────────────────┤
│ 优惠券提示（您有一张满100减10券可用 →）   │
├──────────────────────────────────────┤
│ 商品列表：                               │
│ [勾选框 | 缩略图 | 商品名+规格 | 单价]  │
│ [数量步进控件] [删除按钮（右上角）]      │
├──────────────────────────────────────┤
│ 底部结算栏（固定）：                      │
│ 全选勾选框 + 合计金额 + 去结算（N件）   │
└──────────────────────────────────────┘
```

| 页面 / 模块    | 入口 / 触发      | 内容结构                                                     | 用户操作与系统反馈                                           | 关键规则                                                     |
| -------------- | ---------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 优惠券提示     | 购物车页顶部     | 自动展示用户当前可用的最优券信息                              | 点击跳转领券中心页                                            | 自动匹配用户已领取且未使用的券中抵扣金额最大的一张；无可用券时隐藏 |
| 购物车商品列表 | 购物车页默认展示 | 商品行：勾选框、缩略图、商品名、规格标签、单价、数量控件、删除按钮 | 点击勾选框：单项选中/取消，底部合计实时更新；点击数量控件：数量变化，已勾选时合计同步更新；点击删除：商品从列表移除 | 数量最小值为 1；勾选状态影响合计与结算数量；删除最后一件后进入空状态 |
| 底部结算栏     | 购物车页底部固定 | 全选框 + 「全选」文字 + 合计金额 + 「去结算（N）」按钮       | 全选框：切换所有商品选中状态；点击结算：进入确认订单页                | 结算按钮 N 为当前已勾选商品件数；无选中商品时不触发结算      |
| 空状态         | 购物车无商品时   | 购物车空图标 + 「购物车空空如也」+ 「去选购」按钮        | 点击「去选购」跳转首页                                   | 触发条件：购物车初始为空或删除所有商品后                     |

| 优惠券选择   | 确认订单页默认展示             | 显示可用券张数，已选券名称及抵扣金额                         | 点击进入选券弹窗，选择/取消优惠券                             | 系统默认选中抵扣金额最大的券；不满足门槛的券置灰             |
| 底部提交栏   | 确认订单页底部固定             | 合计金额（已减优惠）+ 「提交订单」按钮                       | 点击提交：调起微信支付                                       | 地址为空时点击提交需提示先填写地址                            |
