## 后台管理 UI 设计说明

> 视觉基调：**专业高效的中后台风格** —— 信息密度高、结构清晰、操作直达。蓝色主色点睛，深色顶栏稳重，大量留白与浅灰分区，让数据与表格成为主角。区别于 C 端的「种草感」，后台追求「一眼看懂、三步操作完成」。

### 全局设计规范（Design Tokens）

#### 色彩

- **主色蓝** `#1677FF`：主按钮、链接、选中态、关键数据、Tab 高亮，是后台点睛色
- **主色浅底** `#E6F0FF`：选中行 / 浅蓝标签底 / hover 底
- **顶栏深色** `#1A2332`：顶部导航栏背景，深色稳重
- **侧栏背景** `#FFFFFF`：侧边导航白底，选中项浅蓝底 + 蓝左条
- **成功 / 正向** `#52C41A`：上架 / 启用 / 正常 / 运行中状态
- **警告** `#FAAD14`：库存不足等预警
- **危险** `#FF4D4F`：下架 / 禁用 / 删除 / 售罄 / 危险操作
- **主标题** `#262626`：页面标题、表格主文字、指标数字
- **正文** `#595959`：常规文字、单元格
- **次要文字** `#8C8C8C`：副信息、表头辅助、时间
- **占位 / 禁用** `#BFBFBF`：placeholder、未选中态
- **页面背景** `#F0F2F5`：中后台经典灰，衬托白卡
- **卡片 / 内容底** `#FFFFFF`：所有卡片、表格、表单纯白
- **表头底** `#FAFAFA`：表格表头浅灰底
- **边框 / 分隔** `#F0F0F0`：表格线、卡片描边、分隔线

#### 字体

- **页面标题** 20px / 600：各功能页主标题
- **区块 / 卡片标题** 16px / 600：卡片标题、表单分组标题
- **表头** 14px / 500：表格列头
- **正文 / 单元格** 14px / 400：表格内容、表单值
- **辅助 / 时间** 12px / 400：副信息、时间戳、说明
- **指标数字** 28-32px / 700：数据看板大数字
- **字体族** -apple-system / PingFang SC / Microsoft YaHei，无衬线

#### 形状与质感

- **卡片圆角**：**8px**（中后台标准，沉稳不浮夸）
- **按钮 / 输入框圆角**：6px
- **标签 Tag 圆角**：4px
- **阴影**：卡片极轻 `0 1px 2px rgba(0,0,0,.06)`；弹窗 `0 6px 16px rgba(0,0,0,.12)`
- **主按钮**：蓝 `#1677FF` 实心 + 白字 + 500
- **次按钮**：白底 + `#D9D9D9` 边框 + `#595959` 字
- **危险按钮**：红 `#FF4D4F` 实心白字，或红字红边（删除类）
- **文字按钮**：表格操作列常用，蓝色「编辑」、红色「删除」，无边框
- **间距**：基础 8px；卡片内边距 16-24px；页面外边距 24px

#### 通用组件规范

- **表格**：白底，表头 `#FAFAFA` 14px 500，行高 54px，分隔线 `#F0F0F0`，行 hover `#FAFAFA`。操作列为文字按钮（蓝/红），右对齐或独立列
- **状态标签 Tag**：圆角 4px 小标签，浅色底 + 同色字：上架/正常 绿、下架/禁用 灰、售罄/异常 红、库存不足 黄、进行中 蓝
- **工具栏**：表格上方一行：左侧搜索框 + 下拉筛选，右侧主操作按钮（如「+ 新增」）。可附「共 N 条」统计
- **分页器**：卡片右下角，当前页蓝色实心，前后翻页箭头，可选每页条数
- **Tab 选项卡**：下划线式，选中蓝字 + 下方 2px 蓝条；状态筛选 Tab 附数量「待发货(8)」
- **表单**：label 左侧 100px 右对齐 + 右侧控件；分组卡片；底部右对齐「重置」次按钮 +「保存」主按钮；保存成功 Toast
- **拖拽排序**：行首 `≡` 手柄，拖拽调整顺序（分类、Banner）
- **危险二次确认**：删除 / 停发 / 确认收货退款等弹出确认框，显示影响范围后再执行

- **主内容区**：背景 `#F0F2F5`，顶部面包屑 14px `#8C8C8C`（末级深色），下方为各功能白卡页面。整体可垂直滚动，内容外边距 24px
- **全局规则**：路由变化时对应菜单自动高亮；折叠动画 200ms；折叠态子菜单 hover 浮层展示

```
├──────────────────────────────────────────────────────┤
│ ┌────────┐ ┌────────┐ ┌────────┐                      │

│ │↑15%绿  │ │↑20%绿  │ │↓10%红  │                      │

├──────────────────────────────────────────────────────┤
│ 待处理事项（有什么活儿要干）                             │
│  待发货 8 红·闪 [发货›]    退款售后 2 红·闪 [去处理›]     │
├──────────────────────────────────────────────────────┤
│ 销售趋势                                    [7天/30天]  │
│   柱状图：日期 × 销售额，柱顶标金额 ¥1,200               │
├──────────────────────────────────────────────────────┤
│ 热销商品 Top 5                              [查看全部 ›]│
│  1  百花蜜(蓝链)  销量 120    ¥10,560                   │
│  2  蜂王浆(蓝链)  销量 88     ¥14,784                   │
│  3  蜂蜡皂(蓝链)  销量 64     ¥2,496                    │
└──────────────────────────────────────────────────────┘
```

- **页面头**：左标题「数据看板」20px 600，无右侧按钮（**不设刷新按钮**，进页即拉最新）。**不设全局时间维度切换**——KPI 卡片本身即「今日」快照，时间维度只由「销售趋势」区块自己掌管，避免两套维度并存造成困惑
- **核心 KPI（一行 3 张）**：白卡圆角 8px，均为**今日固定快照**，回答「赚了多少」——今日销售额 / 今日订单 / 今日新增用户。指标名 14px `#8C8C8C` + 大数字 32px 700 `#262626`，带较昨日环比（↑绿 / ↓红 12px）
- **待处理事项**：一张白卡横排 2 项，回答「有什么活儿要干」——待发货 / 退款售后，数字红 `#FF4D4F` + 闪烁红点，分别带「发货 ›」/「去处理 ›」，点击跳订单管理对应状态 Tab。待付款（等买家付钱）、待收货（等买家确认）店主无需操作，故不列
- **销售趋势**：单张柱状图（日期 × 销售额），**每根柱顶标注当日金额**（如 ¥1,200，14px `#262626`），右上「7 天 / 30 天」切换，默认 7 天——**这是页面唯一的时间维度入口**
- **热销商品 Top 5**：排行表（排名 / 商品名 / 销量 / 销售额），**商品名为蓝色链接样式 `#1677FF`（提示可点击），点击该行跳商品列表并按该商品名预填搜索框定位到该商品**；右上「查看全部 ›」跳商品管理
- **交互**：切换趋势区间重载趋势数据；待处理事项两项点击跳转；热销行点击跳商品列表（搜索框预填商品名）

- **工具栏**：搜索框（商品名）+ 分类下拉 + 状态下拉（上架/下架/售罄）；右侧「共 N 件商品」。筛选实时过滤
- **表格**：列：主图 48px / 名称 / 售价（红 ¥）/ 标签 Tag（取自「标签管理」，**单选，最多一个**）/ 推荐（✓）/ 库存 / 分类 / 状态 Tag / 操作。**库存 < 20 黄色「库存不足」预警，= 0 红色「已售罄」**。操作列：查看（蓝，进只读详情）、编辑（蓝）、上/下架（切换）
- **商品字段**：名称、主图、售价、规格（最多 4）、标签（**单选，可不选**；取自「标签管理」，用于 C 端商品卡片左上角角标）、是否推荐（仅勾选开关，**无需单独写推荐语**；C 端精选推荐卡的种草语自动复用「产品亮点」首条）、**单次限购（0＝不限）**、库存、分类、详情图、产品亮点（多条）
- **营销参与（编辑表单）**：商品编辑表单内**无营销开关**，所有商品默认统一参与满减券与无门槛券。**无任何商品营销 / 折扣开关**
- **分页**：右下角分页器

### 商品编辑弹窗（新增 / 查看 / 编辑）

```

│ 商品名称  [____________________________]            │

│ 单次限购  [___] 件（0 或留空 = 不限购）               │
│ ── 规格与价格（最多 4 个）──                         │
│  规格名 [500g] 售价 [¥88] 库存 [150]  [删除]        │

│                                      [+ 添加规格]   │
│ ── 产品亮点（多条，首条复用为 C 端推荐种草语）──      │
│  · [纯天然零添加_______________]  [删除]            │
│                                      [+ 添加亮点]   │
│ 详情图    [＋上传多张]                               │

└────────────────────────────────────────────────────┘
```

- **弹窗承载**：新增 / 查看 / 编辑统一走同一**弹窗**（不再跳独立页面），居中模态、宽约 640px、内容超高可滚动。商品列表「+ 新增商品」「查看」「编辑」、数据看板热销行点击均打开此弹窗
- **三态差异**：新增＝空表单；**查看＝全字段只读（灰显）+ 底部仅「关闭」**（可一键切到编辑态）；编辑＝预填当前值且可写
- **表单分组**：基础信息（名称 / 主图 / 分类 / 标签 / 推荐 / 单次限购）+ 规格与价格（可增删，最多 4 行：规格名 / 售价 / 库存）+ 产品亮点（多条文本，**首条自动复用为 C 端精选推荐卡种草语**）+ 详情图（多张）
- **无营销开关**：表单内不含任何营销 / 折扣开关
- **底部**：「取消」次按钮 +「保存」主按钮；必填校验（名称 / 主图 / 分类 / 至少 1 条规格）；保存成功 Toast 后关闭弹窗并刷新列表
